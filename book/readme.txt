=============================================
 ダウンロードファイルの使用方法（readme.txt）
=============================================

■ はじめに

ダウンロードファイルには「ArduinoとMATLABで制御系設計をはじめよう！」
の中で使用されている m-file および Simulink モデルが全て含まれています。
本書を片手に，実際に実験を行いながら読み進めることで，理論と実際の両面
から，制御工学の理解を深めることができます。

■ ファイル構成

m-file 及び Simulink モデルは，章別フォルダにまとめられています。ただ
し，第6章と第7章については，同じ Ball and Beam 実験装置を使用している
関係上，同じフォルダにまとめられています。

+ readme.txt          このファイル
+ chap03/             第3章のファイル
+ chap04/             第4章のファイル
+ chap05/             第5章のファイル
+ chap06_and_07/      第6章及び第7章のファイル
+ chap08/             第8章のファイル

■ 動作環境

全てのファイルは，下記の環境で動作することを確認しています。なお，
MATLAB/Simulink については Control System Toolbox が必要です。

OS               Windows 7 32bit
MATLAB/Simulink  R2012a
Arduino          Uno R3 または Mega 2560 R3
Arduino IDE      Version 1.01
ArduinoIO        Version 4.4
